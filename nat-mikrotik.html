LA PREGUNTA

¿Es recomendable hacer NAT en MikroTik?

NAT · Network Address Translation

El NAT traduce direcciones IP privadas a una dirección IP pública para que la red interna pueda salir a internet.

Recordatorio rápido

Muchos equipos internos:

192.168.1.x

➜ NAT ➜

Salen compartiendo:

1 IP pública

Así muchos dispositivos comparten una sola IP pública, lo que ahorra direcciones IPv4, que son un recurso escaso.

✅

Respuesta corta: SÍ

Es una práctica común y recomendada usar el MikroTik como gateway haciendo NAT.

Se logra con una regla srcnat con acción masquerade.

Ejemplo de regla NAT

La regla básica para que toda la red interna salga por la interfaz WAN:

/ip firewall nat
add chain=srcnat out-interface=WAN action=masquerade

Toda conexión que salga por WAN se enmascara con la IP pública del router.

masquerade vs src-nat

masquerade

Ideal para IP dinámica (PPPoE, DHCP). Toma automáticamente la IP de la interfaz. Más simple de configurar.

src-nat to-addresses

Para IP pública fija. Defines la IP de salida manualmente. Ofrece mejor rendimiento al no recalcular la IP en cada conexión.

Ventajas de hacer NAT en MikroTik

🛡️

Firewall integrado en el mismo equipo.

🎛️

Control y flexibilidad total sobre las reglas.

📦

Todo en un equipo: ruteo, NAT y firewall juntos.

⚙️

Fácil de gestionar con Winbox / WebFig.

🔀

Port forwarding (dstnat) para publicar servicios internos hacia internet.

CONTENIDO ADICIONAL

A gran escala: CGNAT

🏢 ¿Qué es CGNAT?

Carrier-Grade NAT: lo usan los ISP cuando faltan IPv4 para muchos clientes. Comparten una IP pública entre varios usuarios.

⚠️ Ojo con la carga

El connection-tracking y miles de conexiones simultáneas cargan la CPU. Hay que dimensionar bien el equipo.

¿Cuándo NO conviene en el MikroTik?

🔥 Router ya saturado de CPU

Si el equipo ya está al límite, agregar NAT puede degradar el rendimiento.

📡 A gran escala

Con muchos clientes se prefiere un equipo de CGNAT / BNG dedicado.

🌐 Se entregan IP públicas directas

Si el cliente recibe una IP pública directa, no hace falta NAT en el router.

Recomendación práctica

🏠

Hogar

NAT en MikroTik es ideal.

🏬

PYME y pequeño ISP

Solución perfecta y económica.

🏗️

Al crecer mucho

Evaluar CGNAT dedicado y dimensionar la CPU.

En resumen: empieza con NAT en MikroTik y escala el equipo conforme crece la red.
