Creación de cuenta en Hetzner

Cómo registrarte paso a paso y crear tu primer servidor en la nube

Hetzner es un proveedor alemán de servidores y nube con una de las mejores relaciones precio / rendimiento del mercado.

¿Qué es Hetzner?

Empresa alemana (fundada en 1997) con centros de datos en Alemania, Finlandia, EE. UU. y Singapur. Ofrece varios productos:

☁️ Hetzner Cloud

VPS por horas/mes. Lo que usaremos aquí.

🖥️ Dedicated

Servidores físicos dedicados.

💾 Storage Box

Almacenamiento y backups.

¿Por qué usar Hetzner?

💲

Económico

Desde menos de 4 € al mes por un VPS.

⚡

CPU rápidas, discos NVMe y red de 1–10 Gbit.

Pagas solo lo que usas, con tope mensual.

Cloud Console clara + API para automatizar.

PASO 1

Ir a la web y registrarse

1

Entra a hetzner.com y pulsa "Cloud" → "Sign Up" / Registrarse.

2

Escribe tu correo electrónico y una contraseña segura.

3

Acepta los términos y pulsa registrar. Revisa tu correo y confirma la cuenta con el enlace.

PASO 2

Datos y verificación de identidad

1

Completa nombre, dirección y país. Deben ser datos reales (se facturan).

2

En cuentas nuevas Hetzner puede pedir verificación de identidad (foto de un documento o selfie) para evitar fraude.

✓

La verificación suele tardar de minutos a unas horas. Hasta aprobarla no podrás crear servidores.

PASO 3

Método de pago

💳

Tarjeta

Crédito o débito (Visa / Mastercard).

🅿️

PayPal

Vincula tu cuenta de PayPal.

🏦

Transferencia

SEPA / banco (según país).

Se factura a fin de mes según el consumo. Puede cobrarse un pequeño importe inicial de validación de la tarjeta.

PASO 4

Crear tu primer servidor

1

Entra a Cloud Console y crea un Proyecto (ej. "Pruebas").

2

Pulsa "Add Server". Elige ubicación (Alemania, Finlandia, EE. UU.…).

3

Elige imagen (Ubuntu, Debian…), el tipo de máquina y añade tu clave SSH.

4

Pulsa "Create & Buy now". En segundos tendrás tu IP pública para conectarte por SSH.

¿Qué máquina elijo?

🧪 Pruebas / 1 servicio pequeño

CX22 o CAX11 (2 vCPU, 4 GB) — perfecto para Uptime Kuma o una web.

🐳 Varios contenedores Docker

CX32 / CAX21 (4 vCPU, 8 GB) — para correr varios servicios a la vez.

🚀 Producción con carga

CPX31 / CPX41 — más CPU AMD y disco para apps con tráfico real.

Recomendaciones

🔑 Usa claves SSH

Más seguro que contraseña; desactiva el login por contraseña.

🔐 Activa 2FA

Protege tu cuenta de Hetzner con doble factor.

💾 Haz backups

Activa backups (~20% extra) o snapshots antes de cambios.

🧱 Firewall en la nube

Usa el Cloud Firewall de Hetzner para abrir solo lo necesario.
