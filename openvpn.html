OpenVPN

Conéctate de forma segura a tu red desde cualquier lugar

OpenVPN es una solución VPN open source que crea un túnel cifrado entre tu dispositivo y la red.

¿Qué es OpenVPN?

Un software que monta una VPN: una "red privada" que viaja cifrada por internet, como si tu equipo estuviera físicamente en la red remota.

🔓

Open source

Gratuito y auditable.

🔒

Cifrado fuerte

Usa SSL/TLS y certificados.

🖧

Multiplataforma

Windows, Linux, Mac, móvil.

Lo descargas y lo entregas

PASO A PASO

Generar el .ovpn

1

Entra a la herramienta web (ovpn.rednuevaconexion.net).

2

Crea un cliente / usuario nuevo y ponle un nombre (ej. juan-laptop).

3

La herramienta genera su certificado y arma el archivo automáticamente.

4

Descarga el archivo .ovpn y entrégaselo al usuario.

Conectarse desde el cliente

🪟

Windows / Mac

Instala OpenVPN Connect e importa el .ovpn.

📱

Android / iOS

App OpenVPN, importa el archivo y conecta.

🐧

Linux

openvpn --config cliente.ovpn

Pulsas Conectar y ya estás dentro de la red de forma segura. 🟢

Seguridad y buenas prácticas

👤 Un certificado por persona

Nunca compartas el mismo .ovpn entre varios.

🚫 Revoca accesos

Si alguien se va, revoca su certificado.

🔥 Firewall

Abre solo el puerto de la VPN (ej. 1194/UDP).

⬆️ Mantén actualizado

Actualiza el servidor y el cliente OpenVPN.
